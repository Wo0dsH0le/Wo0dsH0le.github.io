基础部分

vol.1&2 下注理由&消除理由3

小结

差距在于对自我的提问和正确的回答。

bet/raise reasons：

  1. Value：让更差的牌call。不只是因为你可能有最好的牌。
  2. Bluff：让更好的牌fold。不只是因为你没有希望获胜。

reason-1&2都是源于对方的错误。而人的天性使得更容易犯call的错误，而不是fold的错误。

例：KQo raise，A75r，c-bet一个不会flop对pair fold的松弱对手，why？

bet/raise reasons:

  1. Dead Money：让无论什么牌fold，以获取pot。

解释：

  • 剥削对手底池权益。

    比如6outs。

  • 死钱大到足够补偿被call的损失。

    比如深筹翻前n-bet，大量死钱使得较小成功率即可获利。

  • 对抗更激烈的局中，bluff、边缘牌的投入等效于死钱。

  • 很少作为主要原因，更多是对原因1、2的补充。

    nuts听牌的bet、更少价值面的bet、翻前

保护，是bet的结果，非理由：

QsTs9c-QQ。赢很多牌，保护是需要的，但不是初衷。

Qh9h3c-66，太多听牌（AJs）和66实际上是55开，因为牌力太弱不需要保护，bet主要是为死钱（AT），次要为thin value（A4s）、thin bet（77/88）。

信息，也是bet的结果，非理由：

QT5r-QJ，面对一个松弱对手，bet是有意义的，获得信息的同时，存在被call的value bet。在A22-KK，bet也能获得信息，但不存在value bet。但如果从获取死钱、剥削小手对权益（如果check，小手对的赔率相当于无限好）看，bet是有意义的。

例：AQ vs 88，角色交换，翻前一方all in，都会希望对手fold。

new reason-2:

  • Bluff：为了让对方错误的fold。

具体而言，“错误”的含义是是混合了纯bluff和剥削对方权益两方面。这种情况下，对手是可以依靠底池权益反bluff使我们fold的。

新的reason拓宽了bluff的定义，使得bet不再value/bluff这么两极化，bluff的概念出现在了更多场景中。比如J83-66的raise可能并不坏，我们由此开始了合理的侵略性。

最终bet/raise reasons:

  1. Value：让更差的牌call。不只是因为你可能有最好的牌。
  2. Bluff：为了让对方错误的fold。

思考

从前2章开始，就觉得和已经粗读过的《无限德州扑克进阶指南》可能属于一类书。它们关注从“概念”来理解这个游戏，而不是具体的技巧。有道无术，术尚可求也。

但《easy game》这两章的理解可能没《进阶指南》好，不过德扑本来就是未解决的，不同级别场景的应对也是变化的，不存在绝对的落后、错误。早些时候的书，也有其意义。

我觉得这里的混乱不在于“死钱”这个万金油的辅助砝码的引入，而是“bet/raise理由”这个话题的抽象层次太高了。而德扑中还有形象、位置、筹码量等因素会混合起来影响这个话题。因此，话题的答案注定是一种中庸、参考基准性质的形式。而《easy game》中的理由定义得太绝对了。

甚至改进后的Bluff方面，其实主要就是立足于《进阶指南》中说的“拒绝对手底池权益”，而Value仍比较绝对，相比《进》中“万一获胜时赢得一个较大的底池”，少了对“时间敏感”这一因素的体现。一条街上正确的bet/raise也可能导致后续的棘手局面，但并不意味着之前的决定是错误的。这是德扑“时间敏感”和“随机性”导致的。因此《进》中也采用了一种“贪婪法”的思想，承认后续棘手局面出现的可能性，同时坚持做当前最大EV的决定，当棘手局面出现时，重新评估。

再者，最初value/bluff的理由几乎是“对称的”，但德扑中更多时候是错过翻牌，这就导致了这种“对称”的不正确性。其实后面加入的“拒绝对手底池权益”就是对这种不对称事实的补充。

但《easy game》最后66的事例，指出了侵略性的合理出发点。那就是不同情况下，挑战拒绝对手底池权益的极限。曾经遇到过无脑凶的玩家，有时明明知道他的重注很可能没牌，但是很少有可以抓得很果断的场景，于是被剥削。尽管这是一种损人不利己的剥削，最终被清牌，但明显的，在对手的无脑激进中有不少权益被掠夺。

bet/raise的最好理由，是把《进》中的理由作为参考基准，根据牌桌情况与发展来调整，毕竟“扑克是一门调整的艺术。”

  • 万一获胜时能赢得一个较大的底池
  • 拒绝对手的底池权益

vol.3 翻前手牌范围和翻后胜率

小结

这节没有讨论具体的preflop range，而是指出preflop range构建与postflop策略的联系。

BUT偸盲raise不选择一些很离谱的牌（后面2011年作者补充了反例讨论），主要是因为有些牌postflop的权益太少了，使得postflop策略的侵略性难以被支持。比如K8o-9s7d3c，c-bet被check-call，turn-2s，bb check，BUT陷入尴尬处境。

而选择同花、高张、连张的牌，postflop多了获得足够权益而保持侵略性的可能，甚至bet是听后门花唯一+EV的玩法，尽管不容易命中听牌情况，但相比那些离谱的牌，已经好很多了。

不听花的连张虽然隐蔽，但容易受被同花听面、对手的高张伤害，因为outs少了。权益很多时候不足够支撑侵略性，多是对同花、高张权益的补偿。

保持postflop侵略性的好处有很多，比如赢大底池、bluff对方fold更好的牌、抢死钱、不易被读牌。

进一步对27o这种牌preflop raise思考，这些所谓的“离谱”牌陷入棘手局面，是在于权益不足，不够支撑侵略性，而摊牌获胜后常是不可能。但一方面，preflop raise偸盲的侵略性是固有的，27o陷入postflop棘手局面是因为bluff和对7弱成手，但仍可以止步于preflop获得侵略性收益。另一方面，如果对方保持tight range call preflop，偸盲增长了不摊牌盈利（red line）。这使得对手必须放宽range来对抗27o open，是你用这些牌偸盲-EV。但即使在低级别因为被call得频繁后降低了red line，但你的value bet会得到更好的支付，摊牌盈利增加。

思考

赢得一个底池可简单分成两种情况，一是中牌了，二是打出了弃牌率。从偏价值bet的风格开始思考，preflop如果range太紧，中牌时就太容易被人读出，从而拿不到价值，因而必然引入“平衡”的概念，使得对手放宽range，但这种放宽range背后其实对抗的是平衡range中弱牌的侵略性。

尽管从底池权益来看待更合理，但从range来看更直接一些。因此，平衡或者说侵略性，就是源于基于底池权益的bet，在这种行为之下，所以不管什么range，都具有了“平衡”。

尤其是松与紧的range，其实差异性并不大。在straddle+ante的局，合理的range28-32，不会超过40。翻前的争取有利形势、翻后的有基于位置的bluff或基于底池权益的bet，就是打法的基准。

而且这种思考本身就是着眼至少是一整局牌游戏进程的考虑，明白了它至少不是“-EV”后，需要用自律去贯彻。

vol.4 侵略性和转牌

小结

侵略性（AGG） = 底池权益（PE） + 弃牌率（FE），主要是指turn上second barrel。

QhJh-Th9h2c4d，PE支持；22-843Ar，A带来了FE；As5s-9s7d3cTs，有良好的PE，但是FE降低了。

满足AGG的bet，也是强牌时的bet。但有些情况下，turn上check-raise更好：

  • 对手不太可能有强牌时。Ac5c-8c7c4h，flop对手just call在这个湿面，float range应该是听顺、带对听牌、弱对子、听花、空气。
  • 对手turn上会用一个宽弱的范围bet。8c7c4h2d，听牌、空气bet。

为了防止有时对手在湿面用强牌慢玩，这种check-raise需要一些胜率的支撑。但牌面结构仍是过滤这些强牌，认为对手是用宽弱range bet而c-r的重要因素。比如干面，对手强牌慢玩，turn上raise是合理的。

PE容易计算，FE的计算取决于多方因素：

  • 玩家类型：最重要因素。玩得差的玩家FE很低，不太能fold。对他们我们有一个宽的value bet，但不能用弱手牌来保持AGG。
  • 牌面结构：这是分析FE的关键。简而言之，高张增加了FE，低张、关联张降低了FE。
  • 底池人数：明显的，变相增加了牌面关联性。
  • 形象和感知范围：松、bluff被抓，会降低FE。

但这些因素有时是矛盾的，比如turn A面对松凶玩家，干面多人紧手玩家。

思考

ch3是建立了preflop和postflop（c-bet）的联系，也是引出了agg的概念。这章就是进一步（自然来到turn上）讨论agg这个话题。

对我而言，这一章最重要的点在于，对牌面的思考进行合理的agg。我存在这样的缺点，随着一手牌的进行，容易忽略几条街之间的关联逻辑，尤其是面对重注，陷入抛硬币决策的困境，回想不起之前的逻辑。

比如turn上掉高张，而我没有高张时的bluff。这和flop的牌面、c-bet关联很大。虽然说正常范围中，尤其是open后的pot，大家高张的数量较多，但是flop中低湿面的bet非常能过滤高张的信息，这时候turn上掉高张时的Bluff就不应觉得“我没有那么对手可能正好有”，不合理之处在于，flop对手用什么在call呢？

从这个角度看，现在职业牌手流行的1/3 c-bet是很好的手段。但结合我现在所打的局，大家都是半个买入上桌，死钱偏多，turn上的check-raise适用场景偏少。而底池人数经常偏多，加上低级别，所以pos、thin bet、high value bet更适合一些。

但100bb时，这种second barrel的技巧还是很重要的，也是打法进步的点。

vol.5 下注尺度和薄价值

小结

曾经的落后bet size模式：渐弱bet size。它简单的基于在river较小的bet会被更多的call，这一项思考。

绝大多数玩家的行为，是基于手牌、牌面的。因此bet size可以从极限的角度进行思考，比如1块-n*PSB。

T54J5-JJ/AT/T9：bet时希望得到的call明显是不一样的。这出现了thin value的概念，本质上是为了留下对手最差的call range。

thin value时，明显是存在被更好的牌call的情况，thinness本身就会同时扩大更好/更差的range，但更差range call得更多时，就使得thin value +EV。

thin value本质是保留对手worst call range，因此他的尺度也不是固定的。这里加入了对手风格的考虑。进一步考虑双方风格时，结合筹码量，可能会出现无法thin value的情况。比如3bet底池很大了，之后套池。

thin value不仅同时扩大了“better/worst call range”，对于agg对手，也增大了其bluff的可能。但这只是结果，不是引诱bluff的目的。

思考

thinness其实带来了放大效应，而越是被动的局，就越是保留了更多worst call range，很适合我现在打的环境。而且thin value对多人池、湿面的读牌也有帮助。

因此应该更多的thin value，而不是在自己没有平衡的情况下，极化bet吸引hero call。这其实是对两极化理解的错误。两极化的value很大一部分来自bluff range对对手的剥削，而不是value range对手的支付，value range如果不极化bet对方也会支付，这之间的差量并不大，它更多地意义是平衡bluff range。所以除非我对对手有很特殊的阅读（无脑），或者刚好我形象很差，不然没必要做极化的value bet引诱抓诈，而应该正常的thin value。

而且thin value也是对有时候遇到玩得很好的玩家的试探，可以加入强牌来防御。毕竟对手有技术优势，先考虑防御会更好，选位置也是这个道理。

vol.6 玩家识别和基础读牌

简而言之，不用根据数据识别得太详细，尤其是多开的时候。最重要的是激进和消极，而bad的最大标志是limp，大致定义玩家类型——

  • 差-被动：喜欢call+abc的玩家：多value-bet，不去call他的big bet。
  • 差-激进：喜欢call+不合理bluff：多value-bet，抓bluff。
  • 好-激进：以后章节再谈。
